# opticalflow-opencv3-examples
Quelques exemples d'applications sur les flux optiques avec OpenCV3 et Python 2.7

## Utilisation

* Tracking video:
```bash
cd submodules/tracking
python tracking6.py
```

* Farneback
```bash
cd scripts
python farneback.py
```

* Lucas Kanade
```bash
cd scripts
python lucas_kanade.py
```
